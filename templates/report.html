Colors represent voting tree percentage generated by random forest classifications.

How this value is obtained? Submitted bacterial profiles are mapped along decision trees previously built by random forest to train the classifiers. Each tree gives a classification (roughly: yes/no the mapped data is closed to the model), which all together provides the voting tree percentage. Thus the voting tree percentage translates the degree of similarity between the signatures and the unknown samples. More similar the bacterial signature and the fecal signature is, higher the percentage might be.

For a given source, if the index > 50%, the tested sample obtained the majority of the voting trees, is it considerated to be contaminated by the source. Inversely, if the index < 40%, the tested sample is not considerated to be contaminated by the source. FORENSIC tolerates a margin of error between 40 and 50%.
